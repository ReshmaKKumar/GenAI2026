# GenAI2026
Newsify AI is an intelligent automation system that fetches the latest news, summarizes it using AI, and delivers it directly via email.

This project integrates APIs, AI models (Ollama), and workflow automation (n8n) to create a fully automated news delivery system.

Key Features:

* Fetches real-time news using News API
* Summarizes news using local AI (Ollama – no API cost)
* Formats content with images and links
* Sends automated emails via Gmail
* Runs on scheduled automation

Tech Stack:

* n8n (Workflow Automation)
* Ollama (Local AI Model – llama3)
* NewsAPI (Data Source)
* Gmail API (Email Delivery)

This project demonstrates practical use of AI, APIs, and automation in a real-world application.
